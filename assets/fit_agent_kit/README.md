# Fit Agent — Designer's Branding Kit

Kit completo de ativos SVG para o projeto **Fit Agent** — AI Virtual Assistant for Personal Trainers.
Todos os arquivos estão em formato SVG com **fundo transparente**, prontos para uso em qualquer projeto.

---

## Arquivos incluídos

| Arquivo | Descrição | Uso recomendado |
|---|---|---|
| `01_fitAgent_fullcolor_icon.svg` | Ícone principal — engrenagem com haltere, efeito holográfico full color (azul/ciano) | App icon, splash screen, materiais premium |
| `02_fitAgent_outline_icon.svg` | Ícone minimalista — contorno azul-petróleo, sem preenchimento | Impressão, bordados, uso em fundos coloridos |
| `03_fitAgent_gear_ai_icon.svg` | Ícone decorativo — engrenagem AI/nó de rede | Ícone de seção, favicon, padrão decorativo |
| `04_fitAgent_heartbeat_icon.svg` | Ícone de batida cardíaca com haltere integrado | Banners fitness, redes sociais, assinatura de e-mail |
| `05_fitAgent_mini_gear_icon.svg` | Ícone compacto — engrenagem com haltere e mão | Uso em tamanhos pequenos, ícone de app |
| `06_fitAgent_logo_full.svg` | Logotipo completo — ícone + nome + subtítulo | Cabeçalho de site, apresentações, materiais oficiais |
| `07_fitAgent_wordmark.svg` | Wordmark — apenas texto "FIT AGENT" + subtítulo | Rodapés, assinaturas, uso horizontal |
| `08_fitAgent_outline_dark.svg` | Ícone outline versão escura (azul-marinho) | Fundos claros, papelaria, impressão |
| `09_fitAgent_outline_light.svg` | Ícone outline versão clara (branco) | Fundos escuros, camisetas, banners dark |

## Cores da marca

| Nome | Hex | Uso |
|---|---|---|
| Azul-marinho profundo | `#1a3a4a` | Texto principal |
| Azul-petróleo | `#1a5f7a` | Contornos, subtítulo |
| Ciano vibrante | `#00e5ff` | Destaques holográficos |
| Ciano médio | `#00acc1` | Gradiente principal |
| Ciano claro | `#80deea` | Highlights e brilhos |
| Verde-azulado escuro | `#006064` | Sombras do gradiente |
